# About EasyTag's API



![main](main.png)

EasyTag은 qr태그 스캔을 활용한 기자재 관리 어플리케이션입니다. EasyTag 앱에서 API는 기자재 데이터베이스에 쉽게 접근하여 추가, 수정, 삭제 등의 작업을 간편하게 수행할 수 있도록 DB와의 연동작업을 수행합니다. 본 Repository는 EasyTag 앱에서 사용한 API 파일 및 코드를 포함하고 있습니다.
